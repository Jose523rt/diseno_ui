## diseno_ui
Repositorio que contiene el progreso del desarrollo de la UI utilizada en la aplicación del equipo Kduco

## Reportes de progreso

### Septiembre 25
Se creó el wireframe de la aplicación, procurando solamente establecer la estructura general de la aplicación, usando matiz de grises para tener una idea de la ubicación de cada botón.

### Septiembre 27
Se definieron los colores base de la aplicación, junto con la tipografía. Se preparó para agregar ejemplos de funcionalidades clave (prototipo).

### Septiembre 29
Se implementó el diagrama de flujo de usuario previamente establecido, por medio de botones responsivos y atractivos a la vista. Se implementaron las funcionalidades de agregar producto y una guía para ubicaciones del refrigerador (gráfico real por ser comisionado e implementado).
